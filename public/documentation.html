/featured GET http://localhost:8080/featured JSON N/A
/directors GET http://localhost:8080/directors JSON /:Name
/users GET http://localhost:8080/users JSON /:Username

Users

POST to '/users' to will create a new user.
PUT to 'users/:Username' will allow you to update user information

Expected JSON Format: 


{
    "Username": "user7",
    "Email": "user6@email.com",
    "Password": "user6pass",
    "Birthday": "1985-02-06"
}

POST '/users/:Username/Favorites/push/:MovieID' will add movie to user favorites list
POST '/users/:Username/Favorites/push/:MovieID' will remove movie from user favortes list

Movies

GET '/movies' will return all movies
GET '/movies/:Title' will return movies by title

Directors

GET '/directors' will return all directors
GET '/directors/:Nitle' will return directors by Name
